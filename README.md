# sv-dock workspace

This repository contains `sv-dock`, a Svelte 5 dock layout library with draggable tabs, split panels, floating panels, and serializable layout data.

## Packages

- `packages/sv-dock`: npm package source and demo routes
- `sites/docs`: documentation site workspace

## Development

```bash
pnpm install
pnpm --filter @modif93/sv-dock dev
```

## Checks

```bash
pnpm --filter @modif93/sv-dock check
pnpm --filter @modif93/sv-dock lint
pnpm --filter @modif93/sv-dock test
pnpm --filter @modif93/sv-dock build
```

See [packages/sv-dock/README.md](packages/sv-dock/README.md) for package usage and release checklist details.
